# GrpcKit

A kit for creating gRPC server/client.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'grpc_kit'
```

And then execute:

```
$ bundle
```

Or install it yourself as:

```
$ gem install grpc_kit
```
